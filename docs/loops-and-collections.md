These exercises practice repeated execution, loop conditions, and safe exit conditions.

```python
is_server_down = True
attempts = 0

while is_server_down:
    print("Checking server status...")
    attempts += 1

    if attempts == 3:
        is_server_down = False
        print("Server is back up!")
```
